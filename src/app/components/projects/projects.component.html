AppDelivery & Panel de Administración

Desarrollo de una solución tecnológica destinada a la gestión integral de pedidos a domicilio en el sector de la restauración. La propuesta se compone de dos módulos principales: Aplicación móvil de Delivery: permite a los clientes realizar pedidos de comida a domicilio de forma sencilla, así como consultar el menú disponible del restaurante. Panel de Administración web: herramienta dirigida al personal del establecimiento para la gestión de productos, usuarios y pedidos en tiempo real. Tecnologías utilizadas: El sistema ha sido desarrollado con FlutterFlow, garantizando un diseño multiplataforma, e integrado con Firebase como backend para la gestión de datos, autenticación y servicios en la nube.

Médicas

Backend de Citas Médicas (Spring Boot) API REST para gestionar citas, pacientes y médicos, con operaciones CRUD completas. Desarrollado con Spring Boot, permite integrarse fácilmente con un frontend y maneja la persistencia de datos de forma modular y clara.
